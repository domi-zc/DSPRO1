# Security Policy

## Supported Versions

As this is a student project for a university course, only the current development branch is supported.

| Version | Supported |
| ------- | ------------------ |
| Current `main` branch | :white_check_mark: |
| Older releases / tags | :x: |

## Reporting a Vulnerability

We take the security of our project seriously. If you discover a security vulnerability or potential risk in this repository, please **do not create a public issue**. Public issues allow the vulnerability to be exploited before we can fix it.

### How to Report

Please send an email with the details to:

* **dominic.zybach@stud.hslu.ch**

Please include the following information in your report:
* A description of the vulnerability.
* Steps to reproduce the issue (if possible).

### Our Process

1.  We will acknowledge receipt of your report.
2.  We will investigate the issue and work on a fix.
3.  Once the issue is resolved in the `main` branch, we will notify you.

Thank you for helping keep this project safe!
